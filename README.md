# Pagina statica (upload Aruba)

Questa cartella contiene un esempio di **pagina statica** pronta per essere caricata su Aruba.

## Cosa caricare
Carica i seguenti file nella cartella del tuo hosting Aruba (di solito `www` o `public_html`):

- `index.html`
- `styles.css`
- `script.js`

## Come verificare
1. Apri il pannello Aruba e controlla la **cartella di pubblicazione** (di norma `www`).
2. Carica tutti i file presenti in questa cartella (senza sottocartelle aggiuntive).
3. Apri il tuo dominio nel browser (es. `https://tuodominio.it`).

---

> Se preferisci modificare il contenuto, modifica `index.html` e poi ricarica i file via FTP/SFTP.
